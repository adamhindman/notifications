# Notifications

A vanilla JavaScript project scaffolded with [Vite](https://vite.dev) for fast development with HMR.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server with HMR |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build locally |
